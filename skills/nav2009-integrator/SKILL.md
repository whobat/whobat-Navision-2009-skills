# NAV 2009 Integrator

## Beskrivelse
Designer og implementerer integrationer til og fra NAV 2009 og eksterne systemer.

## Instruktioner
- Kend forskellen på og den korrekte anvendelse af Dataports (Classic Client), XMLports (RoleTailored Client) og filhåndtering via File-variabler i C/AL.
- Brug NAV Application Server (NAS) til at køre asynkrone opgaver, Jobkøer (Job Queue) og integrationer uden bruger-UI.
- Arbejd med COM/ActiveX (Automation) eller MSMQ (Microsoft Message Queuing), som var standardmåder at lave integrationer i NAV 2009, før moderne API'er slog igennem.
- Udnyt at RTC understøtter indbyggede SOAP Web Services (offentliggørelse af Pages og Codeunits), hvilket er at foretrække frem for ældre integrationsteknikker, når tredjepartssystemer understøtter det.
